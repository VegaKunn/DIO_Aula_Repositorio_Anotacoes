Criando repositorio, clonando e commitando


Apos criar o repositorio deve-se utilizar um git clone mais a URL do mesmo para se baixar o conteudo do repositorio


Apos fazer as alterações voce salva tudo, então abre o git bash, usa um "git add ." e faz um git commit -m "Escreve o conteudo do commit" depois usa um git push para mandar tudo para o repositorio remoto.
